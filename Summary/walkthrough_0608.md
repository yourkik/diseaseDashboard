# 🚀 질병 대시보드 지도 고도화 웍스루 (06.08)

사용자님의 피드백을 바탕으로 질병 관리청 데이터 연동을 고도화하고, 에볼라 글로벌 맵 등 다채로운 기능을 성공적으로 구현했습니다. 특히 기존 오픈소스(Leaflet) 맵을 걷어내고 엔터프라이즈급 Azure Maps로 전면 개편했습니다.

## 주요 변경 사항

### 1. 🗺️ 정적 지도 vs 동적 실제 지도 토글 기능
- `map-test/page.js`에 **[기존 버전(정적 다각형)]**과 **[최신 버전(실제 지도+버블 맵)]**을 전환할 수 있는 직관적인 스위칭 UI를 추가했습니다.
- 기존의 휑한 SVG 다각형 지도와 현실적인 지리/도로망을 갖춘 실제 지도를 사용자가 입맛대로 비교하며 볼 수 있습니다.

### 2. 🚀 Azure Maps 기반 엔터프라이즈급 버블 맵 재구축 (`RealMap.jsx`)
- 초기에는 오픈소스인 `Leaflet`(OpenStreetMap)으로 개발했으나, 프론트엔드 성능 및 앱 일관성 강화를 위해 과감히 삭제하고 **Azure Maps SDK**로 전면 교체했습니다. WebGL 하드웨어 가속을 통해 수많은 데이터도 60fps로 부드럽게 렌더링됩니다.
- **절대적 확진자 수 기반 버블 크기**: 지역별 감염자 수치에 따라 버블의 크기가 계산되되, 감염자가 적을 경우 화면을 가리지 않도록 절대적인 캡(Max-size)을 적용했습니다.
- **직관적인 신호등 컬러**: Azure Maps의 Expression 필터를 사용하여 100명 미만은 초록색(안전), 100~499명은 노란색(주의), 500명 이상은 빨간색(심각)으로 자동 렌더링되며 맵 상단에 범례(Legend)를 배치했습니다.
- **상시 텍스트 표시**: 마우스를 올리지 않아도 `SymbolLayer`를 통해 버블 정중앙에 '지역명'과 '확진자 수'가 그림자 효과와 함께 또렷하게 항상 표시됩니다.

### 3. ✨ 전파 경로 애니메이션 (Moving Spark)
- 감염자가 가장 많은 진원지에서 인접 지역으로 뻗어 나가는 선명한 보라색(`Violet`) 실선을 그렸습니다.
- 이에 그치지 않고, `requestAnimationFrame`을 사용하여 보라색 실선 위를 빛의 속도로 달리는 **하얀색 애니메이션 점(Moving Dot)**을 추가 구현함으로써 감염병이 퍼져나가는 '방향성'을 매우 직관적으로 알 수 있게 했습니다.

### 4. 🌍 에볼라 전용 글로벌 맵핑 (Congo)
- 아프리카 콩고민주공화국 구(District) 단위의 `geoBoundaries-COD-ADM2.topojson` 파일을 프론트엔드에 추가했습니다.
- 백엔드에 `ebola_service.py`를 신설하여 HDX 오픈 데이터(`hdx_ebola_...json`)의 지역별 `cases_confirmed`(확진)와 `deaths_suspected`(사망) 수치를 자동으로 집계합니다.
- 사용자가 버튼으로 **'에볼라'** 탭을 선택하면, 지도의 렌즈(center, scale)가 한국에서 **아프리카 콩고**로 부드럽게 넘어가며, 해당 지역의 에볼라 현황이 국내 감염병과 완벽히 동일한 UI/UX(색상 그라데이션)로 그려집니다!

### 5. 최신 데이터 조회 연도 동적 변경 및 질병 매핑
- 하드코딩되어 있던 `2023`년 조회 로직을 제거하고, 서버의 **현재 날짜(`datetime.now()`)를 기준으로 최근 1~2년치 데이터를 병합**하여 가져오도록 백엔드(`disease_stats.py`)를 개선했습니다.
- 프론트엔드에서 직관적으로 **'한타바이러스'** 버튼을 클릭하면, 백엔드는 질병관리청 스펙에 맞는 **'신증후군출혈열'**로 자동 변환하여 데이터를 요청합니다.

### 6. 실시간 뉴스 패널 구축 (GDELT 자동 우회)
- `/api/news` 엔드포인트를 열고 프론트엔드에 글래스모피즘이 적용된 **`NewsPanel.jsx`**를 만들었습니다.
- Bing Search API Key가 없거나 문제가 발생할 경우 500 에러를 뱉고 멈추는 대신, **오픈소스 글로벌 뉴스인 GDELT API로 자동 우회(Fallback)**하여 빈 공간 없이 뉴스를 보여주는 안전망을 구축했습니다.
